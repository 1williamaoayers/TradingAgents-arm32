# 📚 使用说明 - 小白版

## 🎯 这个文件夹是干什么的?

这个文件夹包含了所有用来**测试和使用Google News**的脚本工具。

---

## 📁 文件夹结构(超简单)

```
google_news_scripts/          ← 主文件夹
├── 使用说明.md               ← 你现在看的这个文件
├── README.md                 ← 英文详细文档
├── QUICK_START.sh            ← 快速启动菜单
├── test/                     ← 测试脚本文件夹
│   ├── test_basic.sh         ← 基础测试
│   ├── test_with_dns_fix.sh  ← DNS修复
│   ├── test_complete.sh      ← 完整测试(推荐)
│   └── test_python.py        ← Python测试
└── tools/                    ← 实用工具文件夹
    ├── get_news.sh           ← 一键获取新闻(最常用)⭐
    └── simple_news.py        ← Python工具
```

---

## 🚀 第一次使用(3步搞定)

### 步骤1: 上传到VPS

把整个 `google_news_scripts` 文件夹上传到你的VPS,比如放在 `/root/` 目录下。

```bash
# 在VPS上,进入文件夹
cd /root/google_news_scripts
```

### 步骤2: 运行测试

```bash
# 运行完整测试,看看能不能用
bash test/test_complete.sh
```

如果看到很多 **绿色的✓**,说明成功了!

### 步骤3: 安装Python库

```bash
# 安装需要的库
pip3 install gnews
```

## 💡 日常使用(超简单)

### 方法1: 一键获取新闻(最推荐)⭐

```bash
cd /root/google_news_scripts
bash tools/get_news.sh
```

**就这么简单!** 会自动显示最新的10条新闻。

### 方法2: 使用Python工具

```bash
cd /root/google_news_scripts
python3 tools/simple_news.py
```

会显示头条新闻和科技新闻。

## 🔧 遇到问题怎么办?

### 问题1: DNS解析失败

**症状**: 测试时提示"DNS解析失败"

**解决**:
```bash
bash test/test_with_dns_fix.sh
```

这个脚本会自动帮你修复DNS问题。

### 问题2: Python提示gnews未安装

**症状**: 运行时提示 `ModuleNotFoundError: No module named 'gnews'`

**解决**:
```bash
pip3 install gnews
```

### 问题3: 脚本没有执行权限

**症状**: 提示 `Permission denied`

**解决**:
```bash
chmod +x test/*.sh tools/*.sh QUICK_START.sh
```

## 📋 脚本详细说明

### 🔍 测试脚本(test/ 文件夹)

#### test_basic.sh - 基础测试
- **用途**: 快速检查Google News能不能用
- **什么时候用**: 第一次部署时
- **命令**: `bash test/test_basic.sh`

#### test_with_dns_fix.sh - DNS修复
- **用途**: 自动修复DNS问题
- **什么时候用**: 当DNS解析失败时
- **命令**: `bash test/test_with_dns_fix.sh`
- **特点**: 会自动帮你修复,不用手动操作

#### test_complete.sh - 完整测试 ⭐
- **用途**: 全面测试所有功能
- **什么时候用**: 
  - 第一次部署
  - 定期检查
  - 出问题时排查
- **命令**: `bash test/test_complete.sh`
- **测试内容**: 
  - 网络连接
  - DNS解析
  - Google访问
  - RSS Feed
  - 响应速度
  - Python环境

#### test_python.py - Python测试
- **用途**: 测试Python环境和GNews库
- **什么时候用**: 准备用Python开发时
- **命令**: `python3 test/test_python.py`

### 🛠️ 实用工具(tools/ 文件夹)

#### get_news.sh - 一键获取新闻 ⭐⭐⭐
- **用途**: 最简单的获取新闻方式
- **特点**:
  - 自动检查依赖

## 🎓 常用场景示例

### 场景1: 每天早上看新闻

```bash
cd /root/google_news_scripts
bash tools/get_news.sh
```

### 场景2: 搜索股票新闻

```bash
python3 << 'EOF'
from gnews import GNews
news = GNews().get_news('stock market')
for i, n in enumerate(news[:5], 1):
    print(f"{i}. {n['title']}")
EOF
```

### 场景3: 设置定时任务(每小时自动获取)

```bash
# 编辑定时任务
crontab -e

# 添加这一行(每小时执行一次)
0 * * * * cd /root/google_news_scripts && bash tools/get_news.sh >> /var/log/news.log 2>&1
```

### 场景4: 获取香港新闻

```bash
python3 << 'EOF'
from gnews import GNews
news = GNews(language='zh-Hans', country='HK').get_top_news()
for i, n in enumerate(news[:5], 1):
    print(f"{i}. {n['title']}")
EOF
```

## 📞 需要帮助?

### 检查清单:
- [ ] 网络连接正常吗? (试试 `ping 8.8.8.8`)
- [ ] DNS能解析吗? (试试 `nslookup google.com`)
- [ ] Python安装了吗? (试试 `python3 --version`)
- [ ] gnews库安装了吗? (试试 `pip3 list | grep gnews`)

### 快速诊断:
```bash
bash test/test_complete.sh
```

看输出结果,哪里是 ✗ (红色叉),就是哪里有问题。
